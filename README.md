# 🛍️ MultiShop Marketplace

Современный, быстрый и полностью адаптивный одностраничный интернет-магазин (Landing Page) с тематическим каталогом товаров, динамическим переключением тем оформления и интерактивной формой обратной связи.

Проект создан на чистом стеке (Vanilla Web) без использования тяжелых сторонних фреймворков, что обеспечивает моментальную загрузку и высокую производительность.

---

## ✨ Ключевые возможности сайта

* **🛒 Тематический каталог (3 в 1)**: Ассортимент интернет-магазина структурирован по трем независимым категориям: Одежда 👕, Техника 💻 и Книги 📚. Каждая секция управляется гибкой сеткой CSS Grid.
* **🌗 Выбор темы оформления (Dark / Light)**: 
  * Интегрирована полноценная темная и светлая темы.
  * Переключение происходит мгновенно по кнопке 🌙 / ☀️ в шапке сайта.
  * Выбор пользователя **автоматически сохраняется** в браузере через `localStorage`. При обновлении страницы или повторном закроытии/открытии сайта тема не сбрасывается.
  * Все цветовые переходы анимированы и происходят плавно (`0.25s`).
* **📦 Умная подгрузка товаров («Показать еще»)**: При клике на кнопку новые карточки товаров из JS-массива динамически создаются и автоматически распределяются по своим целевым категориям (`#grid-clothes`, `#grid-tech`, `#grid-books`). После загрузки всех товаров кнопка скрывается.
* **📝 Интерактивная форма обратной связи**: 
  * Отключена стандартная валидация браузера (`novalidate`), проверка данных полностью контролируется скриптом.
  * Реализована кастомная проверка полей (длина имени не менее 2-х символов, проверка email на корректность регулярным выражением).
  * Выводятся понятные сообщения об ошибках, а неверно заполненные поля подсвечиваются.
* **📊 Логирование и сбор данных**: При успешной отправке формы очищенные от пробелов данные пользователя структурировано выводятся в консоль разработчика для последующей интеграции с API.

---

## 🛠️ Технологический стек

* **HTML5**: Семантическая разметка (`header`, `main`, `section`, `article`, `footer`) для улучшения доступности и SEO.
* **CSS3**: Кастомные свойства (переменные) для темизации, Flexbox для позиционирования элементов интерфейса, CSS Grid для адаптивных сеток карточек товаров (`minmax(280px, 1fr)`).
* **Vanilla JavaScript**: Чистый JS для работы с DOM API, обработкой событий (`addEventListener`), валидацией и локальным хранилищем (`localStorage`).

## 📁 Структура репозитория

Для корректной работы сайта на хостинге все файлы находятся в корневой директории:
```text
├── index.html        # Каркас страницы, структура каталога и HTML-форма
├── styles.css        # Цветовые переменные тем, стили, анимации и сетки
└── script.js         # Логика смены тем, валидация полей и подгрузка карточек
```

## ⚙️ Инструкция по локальному запуску

1. Склонируйте репозиторий к себе на компьютер:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в папку с проектом.
3. Откройте файл `index.html` в любом современном браузере двойным щелчком мыши.
4. Чтобы протестировать отправку формы, откройте консоль разработчика в браузере (клавиша `F12` -> вкладка **Console**).

## 🚀 Публикация на GitHub Pages (Деплой)

Сайт полностью готов к бесплатной публикации в сети через **GitHub Pages**:

1. Загрузите файлы проекта в ваш публичный репозиторий на GitHub.
2. Перейдите во вкладку **Settings** (Настройки) вашего репозитория.
3. В левом меню выберите раздел **Pages**.
4. В блоке **Build and deployment** в поле *Source* выберите **Deploy from a branch**.
5. В поле *Branch* выберите ветку `main` (или `master`) и папку `/root`, затем нажмите кнопку **Save**.
6. Через 1-2 минуты страница обновится, и вверху появится зеленая ссылка на ваш работающий сайт вида: `https://github.io`
